# Crypto Analytics Telegram Bot

**Crypto Analytics Bot** — это многофункциональный Telegram-бот для мониторинга криптовалютного рынка, анализа новостей и их влияния на цену биткоина. Бот объединяет данные из нескольких источников (BiBaBot, CoinDesk, DexScreener, CoinGecko), использует машинное обучение для оценки тональности новостей и прогнозирования реакции рынка, предоставляет аналитические метрики в реальном времени и имеет веб-интерфейс для статистики.

## 🚀 Ключевые возможности

- **📰 Новостной агрегатор** – получение последних новостей о биткоине через API [BiBaBot](https://github.com/Hegehub/BiBaBot) (бесплатно, без ключа).
- **🧠 Анализ тональности (sentiment analysis)** – автоматическое определение тональности новости (позитивная/негативная/нейтральная) с помощью встроенного AI BiBaBot.
- **📈 Анализ реакции рынка** – сопоставление тональности новости с изменением цены BTC через 1 час после публикации; выявление новостей, которые совпали с рыночным движением.
- **💰 Рыночные метрики** – интеграция с CoinDesk (цена BTC), DexScreener (ликвидность, объёмы), CoinGecko (доминация BTC), альтернативные источники (индекс страха и жадности).
- **📊 Прогнозирование влияния** – машинное обучение на исторических данных для предсказания, вызовет ли новость движение цены (команда `/predict <url>`).
- **🔔 Подписки** – пользователи могут подписаться на уведомления о новостях, совпадающих с рыночной реакцией (доставка в личные сообщения).
- **👥 Поддержка разных типов чатов**:
  - *Личные чаты* – полное меню с метриками, анализом новостей, подписками.
  - *Группы* – команды для быстрого получения метрик (`/btc`, `/feargreed` и др.).
  - *Каналы* – автоматическая публикация отобранных новостей, совпадающих с реакцией рынка.
  - *Администраторы* – панель управления, ручная публикация, статистика.
- **🗄 Хранение данных** – SQLite (или PostgreSQL) для истории новостей, пользователей, подписок.
- **⚡ Кэширование** – Redis для ускорения доступа к ценам и состояниям.
- **🌐 Веб-интерфейс** – FastAPI + Chart.js для визуализации статистики (количество новостей, точность совпадений и др.).
- **⏰ Планировщик задач** – APScheduler для периодической проверки новостей, обновления цен, анализа совпадений.

## 🛠 Технологии

- **Язык**: Python 3.10+
- **Telegram Bot**: aiogram 3.x
- **Асинхронные HTTP-запросы**: aiohttp
- **База данных**: SQLAlchemy 1.4+ (aiosqlite / asyncpg для PostgreSQL)
- **Кэш**: Redis (aioredis)
- **Планировщик**: APScheduler
- **Веб-интерфейс**: FastAPI, Jinja2, Uvicorn
- **Машинное обучение**: scikit-learn (для прогноза влияния)

## 📦 Установка и запуск

### Предварительные требования
- Python 3.10+
- Redis сервер (локально или удалённо)
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

### Шаги

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/yourusername/crypto-bot.git
   cd crypto-bot
```

1. Установить зависимости
   ```bash
   pip install -r requirements.txt
   ```
2. Настроить переменные окружения
      Создайте файл .env:
   ```env
   BOT_TOKEN=ваш_токен_бота
   CHANNEL_ID=@имя_канала_для_публикаций
   ADMIN_IDS=123456789,987654321  # ID администраторов через запятую
   REDIS_URL=redis://localhost:6379/0
   DATABASE_URL=sqlite+aiosqlite:///crypto_bot.db  # или postgresql+asyncpg://...
   ```
3. Запустить Redis (если локально)
   ```bash
   redis-server
   ```
4. Инициализировать базу данных
      При первом запуске бота таблицы создадутся автоматически.
5. Запустить бота
   ```bash
   python bot.py
   ```
6. Запустить веб-интерфейс (в отдельном терминале)
   ```bash
   uvicorn web:app --reload --port 8000
   ```

Теперь бот доступен в Telegram, веб-интерфейс по адресу http://localhost:8000.

🤖 Использование бота

Личные чаты

· /start – главное меню с кнопками:
  · 📊 BTC Price – текущая цена биткоина
  · 😨 Fear & Greed – индекс страха и жадности
  · 📈 Dominance – доминация BTC
  · 💧 Liquidity – данные о ликвидности с DexScreener
  · 📰 Analyze News – анализ новости по ссылке
  · 🔔 Subscribe – подписка на уведомления о совпадающих новостях

Групповые чаты

· /btc – цена BTC
· /feargreed – индекс страха и жадности
· /dominance – доминация BTC
· /liquidity – данные о ликвидности
· /predict <url> – прогноз влияния новости на цену

Каналы

· Бот автоматически публикует новости, которые совпали с рыночной реакцией (настраиваемая периодичность).

Администраторы

· /admin – открыть админ-панель
  · 📢 Post to Channel – ручная публикация сообщения в канал
  · 📊 Bot Stats – статистика (пользователи, новости, совпадения)
  · 👥 Subscribers – список подписчиков

📡 API (веб-интерфейс)

Веб-интерфейс доступен по адресу http://localhost:8000 и предоставляет:

· Общую статистику (количество новостей, совпадений, пользователей).
· Графики динамики совпадений и точности прогнозов.
· API эндпоинты для получения данных в JSON (например, /api/stats).

🧠 Машинное обучение

Модуль ml_model.py:

· Обучает модель (логистическая регрессия) на исторических данных из БД: признаки – тональность новости, время суток, источник; целевая переменная – изменилась ли цена >0.5% через 1 час.
· Модель переобучается периодически по расписанию.
· Команда /predict <url> возвращает вероятность того, что новость вызовет движение цены.
